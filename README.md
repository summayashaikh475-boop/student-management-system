🎓 Student Management System

A simple and efficient Student Management System built using Python.
This project demonstrates the use of Object-Oriented Programming (OOP), file handling, and a command-line interface (CLI).

🚀 Features

➕ Add new students
📋 View all students
🔍 Search student by name
✏️ Update student marks
❌ Delete student
📊 Calculate average marks
🏆 Show toppers (sorted by marks)
💾 Save data to file
📂 Load data from file

🛠️ Technologies Used

Python
Object-Oriented Programming (OOP)
File Handling

▶️ How to Run the Project

Download or clone this repository
Open terminal in the project folder
Run the program:
python main.py

📂 Project Structure

student-management-system/
│── main.py
│── students.txt (auto-created)
